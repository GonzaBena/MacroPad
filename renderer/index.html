Cómo funciona

El Arduino envía texto por Serial.println(). Cada línea es una señal. Acá definís qué hace la PC cuando recibe cada una.

Código Arduino de ejemplo
// Declará el pin del botón
const int BTN = 2;

void setup() {
  Serial.begin(9600);
  pinMode(BTN, INPUT_PULLUP);
}

void loop() {
  if (digitalRead(BTN) == LOW) {
    Serial.println("BUTTON_PRESSED");
    delay(200); // debounce
  }
}
